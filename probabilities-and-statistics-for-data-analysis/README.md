## Probabilities and Statistics for Data Analysis

This folder contains two projects focused on statistical modeling, simulation, and inference using R.

### Project 1 – Distribution Simulation and Statistical Analysis

Implements simulation of discrete and continuous random variables and analyzes their statistical properties.

**Techniques & Concepts:**
- Random variable simulation (Poisson, Exponential)
- Law of Large Numbers (effect of sample size)
- Empirical vs theoretical distributions
- Summary statistics (mean, standard deviation, median, IQR)
- Quantile estimation (sample vs theoretical)
- Distribution visualization (histograms, density plots)

**Libraries & Tools:**
- Base R (rpois, rexp, quantile, density)
- Data visualization (hist, barplot)

---

### Project 2 – Statistical Inference and Confidence Intervals

Applies statistical inference methods on real-like data using hypothesis testing and confidence intervals.

**Techniques & Concepts:**
- Central Limit Theorem (CLT)
- Confidence intervals for mean
- t-test for small samples
- Proportion estimation and confidence intervals
- Hypothesis testing for binary outcomes

**Libraries & Tools:**
- Base R (t.test, prop.test)

---

### Summary

These projects focus on building a strong foundation in:
- Statistical thinking
- Simulation-based analysis
- Inferential statistics
- Data-driven reasoning using R
