# LLMs with Knowledge Graphs

This course focuses on combining Large Language Models (LLMs) with Knowledge Graphs to build structured and intelligent systems.

## Overview

Explores how structured data (knowledge graphs) and unstructured data (LLMs) can be integrated for tasks such as information retrieval, reasoning, and question answering.

## Topics Covered

- Knowledge Graph design and modeling
- Graph databases (Neo4j)
- Querying with Cypher
- Integration of LLMs with structured data
- Graph-based reasoning and retrieval

## Projects

- Design and implementation of a Knowledge Graph using Neo4j
- Querying and extracting information using Cypher
- Building a system for structured data exploration and question answering

## Tools & Technologies

Neo4j • Cypher • Python • LLM frameworks
